Configuration¶

The Acme library is configured through a Config object passed to the constructor.

Basic Setup¶

The simplest configuration requires only an API key:

from acme import Client

client = Client(api_key="sk-...")

Advanced Options¶

For production deployments, consider these additional options:

client = Client(
    api_key="sk-...",
    max_retries=3,
    timeout=60,
    base_url="https://api.acme.dev",
)
class Config

Configuration object for the Acme client.

Parameters:
  • api_key (str) – Required. Your Acme API key.
  • max_retries (int) – Number of retries on transient failures. Default: 2.
  • timeout (float) – Request timeout in seconds. Default: 30.

Environment Variables¶

All config values can be set via environment variables:

VariableConfig KeyDescription
ACME_API_KEYapi_keyAPI authentication key
ACME_BASE_URLbase_urlOverride API endpoint
ACME_TIMEOUTtimeoutRequest timeout (seconds)
