Create User

Creates a new user account. Returns the created user object on success.

HTTP Request

POST https://api.acme.dev/v1/users

Authentication

This endpoint requires a valid API key passed in the Authorization header.

Request Body

ParameterTypeRequiredDescription
emailstringYesThe user's email address. Must be unique.
namestringYesFull display name.
rolestringNoOne of admin, member. Defaults to member.
metadataobjectNoArbitrary key-value metadata.

Example Request

curl -X POST https://api.acme.dev/v1/users \
  -H "Authorization: Bearer $ACME_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "alice@example.com",
    "name": "Alice Smith",
    "role": "admin"
  }'

Response

{
  "id": "usr_01h8xqzk5p",
  "email": "alice@example.com",
  "name": "Alice Smith",
  "role": "admin",
  "created_at": "2024-01-15T10:30:00Z",
  "metadata": {}
}

Error Codes

StatusCodeDescription
400invalid_requestMissing or malformed request body.
401unauthorizedMissing or invalid API key.
409email_takenAn account with this email already exists.
422validation_errorOne or more fields failed validation.
