Installation

Get started with the Acme SDK in minutes. This guide covers installation, initial setup, and your first API call.

Prerequisites

Before installing, make sure you have:

  • Node.js 18 or higher
  • An Acme API key (get one at dashboard.acme.dev)
  • npm, yarn, or pnpm

Install the Package

Install the SDK using your preferred package manager:

npm install @acme/sdk

Or with yarn:

yarn add @acme/sdk

Initialize the Client

Create a new client instance with your API key:

import { AcmeClient } from '@acme/sdk';

const client = new AcmeClient({
  apiKey: process.env.ACME_API_KEY,
  timeout: 30000,
});

Configuration Options

The client accepts the following options:

Option Type Default Description
apiKey string — Your Acme API key. Required.
timeout number 10000 Request timeout in milliseconds.
baseUrl string https://api.acme.dev Override the API base URL.

Your First Request

Once initialized, you can make API calls:

const result = await client.process({
  input: 'Hello, world!',
  model: 'acme-v2',
});

console.log(result.output);
